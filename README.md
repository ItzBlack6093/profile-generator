# About 'mode-icon' branch

This branch will generate with 6 modes, along with its icon

+ Tetra League
+ 40 Lines
+ Blitz
+ Quick Play
+ Expert Quick Play
+ Zen (Score and Level (Symbols))

...but it will be 2x3 instead of normal 3x2
